Ans: Javascript Programming language is a scripting Language. We can update, change, delete, animate any kind of content with the help of javascript language dynamically. Javascript has a document object model (DOM) which gives us the ability to change elements and select the node and manipulate. javascript runs with help of the Chrome V8 engine coded with c++, it turns our language with machine language which follows the ECMAScript standard. By nature, javascript cant run multiple operations at a time. That's why javascript is called a single-threaded programming language.

Web storage API is used to store data into the web browser. There are two different ways to store the data. One is session storage and the other is local storage. The Data store in session storage means the data is persisted for a particular time . We Can store a minimum 5MB data in session storage. In other hands If we go for Local storage it never expires. For example, if we shut down the browser or the system and come back again the data will remain persisted in the same browser
